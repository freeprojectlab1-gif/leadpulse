# Email Scraping Target Ideas

Ye file un keywords, cities, aur social platforms ki list hai jahan se sabse zyada "No Website" ya direct contact waale emails milte hain.

## 🎯 Top Categories (Keywords)
1. **Interior Designer** - Hamesha ig/facebook par active rehte hain, par sabke paas website nahi hoti.
2. **Real Estate Agent / Broker** - Personal emails easily milte hain social media par.
3. **Plumber / Electrician / Hardware Shop** - Local businesses jinki online presence mostly Facebook tak limited hoti hai.
4. **Restaurant / Cafe / Bakery** - Naye cafes mostly Instagram use karte hain promotion ke liye.
5. **Dentist / Clinic / Hospital** - Chote clinics ki hamesha website nahi hoti.
6. **Fashion Boutique / Clothing Store** - Instagram unka main platform hota hai.
7. **Lawyer / Advocate / Law Firm** - Independent lawyers direct email share karte hain.
8. **Fitness Trainer / Gym** - Instagram par leads easily convert hote hain.
9. **Event Planner / Wedding Photographer** - Visual portfolio IG/FB par hota hai, website kam hoti hai.

## 🌍 Top Places (Cities)
1. **Dubai / Abu Dhabi** (High conversion, modern businesses without sites)
2. **New York / Dallas / Texas / Austin** (USA local shops)
3. **London / Manchester** (UK local services)
4. **Mumbai / Delhi / Bangalore / Pune** (Indian local businesses like Hardware, Fashion)
5. **Toronto / Vancouver** (Canada)
6. **Sydney / Melbourne** (Australia)
7. **Singapore**

## 📱 Best Social Platforms (Sources)
1. **Instagram (ig)** - Sabse best source "Fashion", "Food", "Design", aur "Fitness" categories ke liye.
2. **Facebook** - "Plumber", "Hardware", "Real Estate", aur local services ke liye best.
3. **LinkedIn** - B2B leads ke liye best (Software, Marketing, Corporate lawyers).
4. **Google Maps (map)** - Local shops (Hardware, Clinic) ka verified data nikalne ke liye best.

---
**💡 Pro Tip for "No Website" filter:**
Hamesha `ig` aur `facebook` source select karein aur `no_website` filter on rakhein jab aap *Interior Designer in Dubai* ya *Plumber in New York* search kar rahe hon.
